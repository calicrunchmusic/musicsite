CaliCrunchMusic.com static website starter

How to use:
1. Upload these files to your hosting folder.
2. Replace placeholder links in music.html and links.html.
3. Add real images/videos/audio inside /assets.
4. Optional assets:
   - assets/bg-studio.jpg
   - assets/site-music.mp3
5. Open index.html in a browser to test.

Pages included:
index.html, music.html, bio.html, gallery.html, merch.html, links.html, contact.html, videos.html, style.css, script.js
